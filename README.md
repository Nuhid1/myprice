# MyPrice

**MyPrice** is a smart price comparison platform that helps users find the best product prices across multiple online stores in Bangladesh.

Instead of searching multiple e-commerce sites manually, MyPrice aggregates product listings and allows users to compare prices instantly.

---

## 🌐 Live Website

https://mypricebd.netlify.app

---

## 🚀 Features

- 🔎 Search products across multiple online stores  
- 💰 Compare prices from different sellers  
- ⚡ Fast and simple search experience  
- 🛒 Direct links to the original stores  
- 📊 Aggregated product results  

---

## 🧠 How It Works

1. The user searches for a product.
2. The backend fetches product data from multiple online stores.
3. Data is normalized and stored.
4. Results are displayed in a unified comparison interface.
